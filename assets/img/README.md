Este directorio contiene imágenes usadas por el portfolio.

- Reemplazá los archivos listados en `js/data.js` por tus imágenes reales.
- Nombres sugeridos:
  - about-vertical.jpg (imagen vertical para la sección About)
  - video-thumb-1.jpg, video-thumb-2.jpg, video-thumb-3.jpg (thumbnails)
  - photo-1.jpg ... photo-3.jpg (fotografías editoriales)

Simplemente copia tus archivos con esos nombres o editá `js/data.js` para usar otros nombres.
