Directorio para videos (opcional).

- Para videos locales: poné los archivos en esta carpeta y actualizá `videos[].src` en `js/data.js` (ej: 'assets/video/reel1.mp4').
- Para externos: usá URLs de YouTube/Vimeo en `videos[].src` (ej: 'https://www.youtube.com/embed/...').
- Nota: videos locales hacen el repo más grande; considerá subir a un CDN si son pesados.
- Recomendado: H.264 MP4 o WebM optimizados para web y versión corta para reels (9:16).
